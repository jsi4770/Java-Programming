# Java Programming 학습 아카이브

이 저장소는 Java 프로그래밍 수업에서 학습한 내용과 실습 코드, 객체지향 개념 정리 및 프로젝트를 기록하기 위한 공간입니다.

## 📌 목적
- Java 문법 및 객체지향 프로그래밍(OOP) 이해
- 프로그램 설계 능력 향상
- 학습 과정 기록 및 포트폴리오 활용

## 📂 폴더 구조

- notes/  
  → Java 이론 및 개념 정리

- exercises/  
  → 수업 실습 및 주차별 연습 코드

- projects/  
  → 간단한 프로젝트 및 응용 프로그램

## 📚 학습 내용

- 기본 문법
- 클래스와 객체
- 생성자
- 상속
- 다형성
- 인터페이스
- 컬렉션 프레임워크

## 💡 학습 방식

- 수업 내용을 기반으로 직접 코드를 작성하고 정리
- 핵심 개념은 notes에 문서화
- 실습 코드는 exercises에 주차별로 정리
- 프로젝트는 별도로 관리

## 🚀 목표

객체지향 프로그래밍의 핵심 개념을 이해하고,  
구조적으로 프로그램을 설계할 수 있는 능력을 기르는 것을 목표로 합니다.
