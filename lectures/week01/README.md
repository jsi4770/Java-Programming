## 📌 Week 01 - 프로그램과 C 언어

### 1. 프로그램과 프로그래밍

- **프로그램**  
  → 컴퓨터가 특정 작업을 수행하도록 지시하는 명령어의 집합  

- **프로그래밍**  
  → 프로그램을 만드는 과정 (문제 해결을 위한 명령 작성)

---

### 2. 프로그래밍 언어

- 컴퓨터에게 명령을 전달하기 위한 언어  

#### ▪ 언어의 종류
- **기계어**: 0과 1로 구성 (컴퓨터만 이해 가능)
- **어셈블리어**: 기계어를 사람이 이해하기 쉽게 표현
- **고급 언어**: 사람이 이해하기 쉬운 언어 (C, Java, Python 등)

#### ▪ 실행 방식
- **컴파일러**  
  → 전체 코드를 기계어로 변환

- **인터프리터**  
  → 코드를 한 줄씩 실행

---

### 3. C 언어 개요

- 1972년 AT&T 벨 연구소에서 개발  
- 개발자: 데니스 리치  
- 유닉스 운영체제 개발을 위해 만들어짐  

---

### 4. C 언어의 특징

- **범용 프로그래밍 언어** → 다양한 분야에서 사용 가능  
- **저급 + 고급 언어의 장점 결합**
  - 하드웨어 제어 가능
  - 개발 편의성 확보
- **간결한 표현 가능**
- **이식성이 좋음** (여러 환경에서 실행 가능)

---

### 5. 프로그램 개발 과정

1. 요구사항 분석  
2. 설계  
3. 구현 (코딩)  
4. 테스트  
5. 유지보수  

---

### 6. C 프로그램 실행 과정

1. 소스코드 작성 (.c 파일)  
2. 컴파일 (기계어로 변환)  
3. 링크 (실행 파일 생성)  
4. 실행 및 디버깅
